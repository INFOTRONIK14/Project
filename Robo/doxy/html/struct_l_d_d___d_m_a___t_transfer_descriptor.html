Detailed Description

Type specifying the DMA Transfer descriptor information structure.

Field Documentation

LDD_DMA_TAddressOffset LDD_DMA_TTransferDescriptor::AddressOffset

Address offset value. Address specified in AfterRequestComplete item is adjusted by stored value. See the LDD_DMA_TAfterRequest type declaration.

LDD_DMA_TAfterRequest LDD_DMA_TTransferDescriptor::AfterRequestComplete

Type of an action after the elemental read/write operation is done. For the description of allowed values see the LDD_DMA_TAfterRequest type declaration.

LDD_DMA_TAfterTransfer LDD_DMA_TTransferDescriptor::AfterTransferComplete

Type of an action executed after the last transfer operation is done. For the description of allowed values see the LDD_DMA_TAfterTransfer type declaration.

LDD_DMA_TBandwidthControl LDD_DMA_TTransferDescriptor::BandwidthControl

DMA channel bandwidth control. See the DMA_PDD header file for detail description of allowed values.

LDD_DMA_TByteCount LDD_DMA_TTransferDescriptor::ByteCount

Size of data in bytes to be transferred in single transfer.

bool LDD_DMA_TTransferDescriptor::ChannelAutoSelection

TRUE - DMA channel autoselection engine is used. FALSE - DMA fixed channel is used.

bool LDD_DMA_TTransferDescriptor::ChannelEnabled

TRUE - DMA channel is allocated and used by DMATransfer component.

LDD_DMA_TChannelNumber LDD_DMA_TTransferDescriptor::ChannelNumber

If ChannelAutoSelection is FALSE this item contains fixed channel number. If ChannelAutoSelection is TRUE then after allocation this item is filled by autoselected channel number.

LDD_DMA_TAddress LDD_DMA_TTransferDescriptor::DestinationAddress

Address of a DMA transfer destination.

LDD_DMA_TAddressOffset LDD_DMA_TTransferDescriptor::DestinationAddressAdjustment

Destination address adjustment value. Used only if the AfterTransferComplete item is set to LDD_DMA_ADDRESS_ADJUSTMENT.

LDD_DMA_TAddressOffset LDD_DMA_TTransferDescriptor::DestinationAddressOffset

Offset to be added to the destination address after each elemental write operation.

LDD_DMA_TModuloSize LDD_DMA_TTransferDescriptor::DestinationModuloSize

Destination address modulo size. For the description of allowed values see the LDD_DMA_TModuloSize type declaration.

LDD_DMA_TTransferSize LDD_DMA_TTransferDescriptor::DestinationTransferSize

Destination data transfer size (size of a elemental write operation). See the DMA_PDD header file for detail description of allowed values.

bool LDD_DMA_TTransferDescriptor::DisableAfterRequest

TRUE - DMA transfer request is automatically disabled after transfer complete.

bool LDD_DMA_TTransferDescriptor::InnerLoopChannelLink

TRUE - Inner loop channel linking enabled (if the nested operation is used, then this item enables the minor (inner) loop channel linking).

LDD_DMA_TChannelNumber LDD_DMA_TTransferDescriptor::InnerLoopLinkedChannel

Linked DMA channel number (used only if the InnerLoopChannelLink item is set TRUE)

bool LDD_DMA_TTransferDescriptor::Interrupts

TRUE - interrupts are requested.

bool LDD_DMA_TTransferDescriptor::OnComplete

TRUE - event is enabled during initialization.

void(* LDD_DMA_TTransferDescriptor::OnCompleteEventPtr)(LDD_TUserData *UserDataPtr)

Pointer to the OnComplete event, NULL if event is not used.

bool LDD_DMA_TTransferDescriptor::OnError

TRUE - event is enabled during initialization.

void(* LDD_DMA_TTransferDescriptor::OnErrorEventPtr)(LDD_TUserData *UserDataPtr)

Pointer to the OnError event, NULL if event is not used.

bool LDD_DMA_TTransferDescriptor::OnHalfComplete

TRUE - event is enabled during initialization.

bool LDD_DMA_TTransferDescriptor::OuterLoopChannelLink

TRUE - Outer (major) loop channel linking is enabled. Enables channel linking after transfer completes.

LDD_DMA_TOuterLoopCount LDD_DMA_TTransferDescriptor::OuterLoopCount

Number of the outer loop iteration in the Nested operation mode, otherwise should have value of one.

LDD_DMA_TChannelNumber LDD_DMA_TTransferDescriptor::OuterLoopLinkedChannel

Outer (major) loop linked DMA channel number (used only if the OuterLoopChannelLink item is set TRUE).

bool LDD_DMA_TTransferDescriptor::PeriodicTrigger

TRUE - periodic trigger is required, FALSE - periodic trigger is not required.

LDD_DMA_TAddress LDD_DMA_TTransferDescriptor::ScatterGatherAddress

Scatter / gather address. Used only if the AfterTransferComplete item is set to LDD_DMA_SCATTER_GATHER.

LDD_DMA_TAddress LDD_DMA_TTransferDescriptor::SourceAddress

Address of a DMA transfer source data.

LDD_DMA_TAddressOffset LDD_DMA_TTransferDescriptor::SourceAddressAdjustment

Source address adjustment value. Used only if the AfterTransferComplete item is set to LDD_DMA_ADDRESS_ADJUSTMENT.

LDD_DMA_TAddressOffset LDD_DMA_TTransferDescriptor::SourceAddressOffset

Offset to be added to the source address after each elemental read operation.

LDD_DMA_TModuloSize LDD_DMA_TTransferDescriptor::SourceModuloSize

Source address modulo size. For the description of allowed values see the LDD_DMA_TModuloSize type declaration.

LDD_DMA_TTransferSize LDD_DMA_TTransferDescriptor::SourceTransferSize

Source data transfer size (size of a elemental read operation). See the DMA_PDD header file for detail description of allowed values.

LDD_DMA_TTransferMode LDD_DMA_TTransferDescriptor::TransferMode

Selects DMA transfer mode. For the description of allowed values see the LDD_DMA_TTransferMode type declaration.

LDD_DMA_TTriggerSource LDD_DMA_TTransferDescriptor::TriggerSource

Trigger source number. For the description of allowed values see the LDD_DMA_TBTriggerType declaration.

LDD_DMA_TTriggerType LDD_DMA_TTransferDescriptor::TriggerType

DMA transfer trigger type. For the description of allowed values see the LDD_DMA_TBTriggerType declaration.

LDD_TUserData* LDD_DMA_TTransferDescriptor::UserDataPtr

User device data structure pointer to be returned by the DMA_LDD component's ISR to the dynamic callback of this transfer descriptor.
